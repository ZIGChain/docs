# ZIGChain RPC and API Endpoints

These endpoints are provided for developers to test their applications and integrations. They are not recommended for production use. For production, we recommend running your own full node.

## Testnet

| **Service** | **Address**                                   |
| ----------- | --------------------------------------------- |
| RPC         | https://public-zigchain-testnet-rpc.numia.xyz |
| API         | https://public-zigchain-testnet-lcd.numia.xyz |
| Faucet      | https://faucet.zigchain.com                   |

## Mainnet

| **Service** | **Address**                           |
| ----------- | ------------------------------------- |
| API         | https://public-zigchain-lcd.numia.xyz |
| RPC         | https://public-zigchain-rpc.numia.xyz |
